# 🎨 CustOmnivox - Dark mode

Une extension Chrome qui applique un thème sombre personnalisé à toutes les plateformes Omnivox (Intr, Léa, MIO, ESTD, etc.).

![Version](https://img.shields.io/badge/version-1.1-blue)
![Licence](https://img.shields.io/badge/licence-MIT-green)
![Chrome](https://img.shields.io/badge/chrome-extension-orange)

## ✨ Fonctionnalités

- 🌙 **Thème sombre** sur toutes les pages Omnivox
    - Accueil (`/intr/`)
    - Page de connexion (`/Login/Account/`)
    - Léa (`lea.omnivox.ca`)
    - Dépôt de travaux (`/cvir/dtrv/DepotTravail.aspx`)
    - 2FA / MFA (`/apps/mfa/login`)
    - ESTD (`estd.omnivox.ca`)
    - MIO (`/WebApplication/Module.MIOE/`)
